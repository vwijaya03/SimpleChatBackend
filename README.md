## Installation

```bash
$ docker pull mongo
$ docker run -d -p 27017:27017 --name mongo mongo
$ git clone https://github.com/vwijaya03/SimpleChatBackend.git
$ npm install
$ npm start
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
